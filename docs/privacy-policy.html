Politique de confidentialité — MangaCentral

Dernière mise à jour : 8 avril 2026

MangaCentral est une extension de navigateur (Chrome / navigateurs compatibles Chromium) qui aide au suivi de lecture et propose des outils d’assistance sur les pages que vous consultez. Cette page décrit quelles données sont traitées, où elles sont stockées et quels services tiers peuvent être sollicités lorsque vous utilisez l’extension.

1. Responsable et contact

Pour toute question relative à cette politique, vous pouvez contacter l’éditeur de l’extension à l’adresse suivante : contact@example.com.

2. Données stockées localement sur votre appareil

L’extension utilise le stockage local du navigateur (chrome.storage.local) pour :

  • votre historique de lecture (titres, numéros de chapitre, URL des pages visitées dans ce cadre, éventuellement liens de couverture) ;
  • votre bibliothèque et paramètres (vitesse de défilement, sites personnalisés, etc.) ;
  • si vous le choisissez, une clé API Google Gemini que vous saisissez vous-même dans les paramètres — elle reste sur votre machine et n’est pas envoyée à nos serveurs (nous n’exploitons pas de serveur dédié à l’extension pour la collecter).

Ces données ne sont pas synchronisées sur nos systèmes : elles restent sur l’appareil où l’extension est installée, sauf export ou synchronisation que vous feriez vous-même en dehors de l’extension.

3. Capture d’écran et image pour traduction / OCR

Lorsque vous déclenchez une fonctionnalité de type capture ou sélection de zone, l’extension peut capturer la zone visible de l’onglet concerné ou traiter une image issue de cette sélection. Ces actions ne s’exécutent que sur votre initiative.

Si vous avez configuré une clé API Gemini, l’image (ou une représentation dérivée) peut être envoyée aux serveurs Google (API Gemini) uniquement pour produire une transcription / traduction demandée par vous. Nous ne recevons pas une copie de ces images sur un serveur « MangaCentral ».

4. Recherche et métadonnées (APIs publiques)

Pour la recherche de titres et suggestions de liens, l’extension peut appeler des services tiers publics, notamment :

  • MangaDex (API api.mangadex.org) ;
  • Jikan (API api.jikan.moe) ;
  • AniList (GraphQL graphql.anilist.co).

Ces appels portent en principe sur des requêtes de recherche (titres) et la récupération de métadonnées publiques. Les conditions et politiques de confidentialité de ces services s’appliquent en complément.

Si une clé Gemini est configurée, une suggestion de lien peut également être obtenue via l’API Gemini (texte), selon les conditions de Google.

5. Pas d’enregistrement de chapitres sur le disque

L’extension ne propose pas de téléchargement de chapitres ou d’archives vers votre ordinateur. Les images de page peuvent être traitées en mémoire uniquement pour les actions que vous lancez (par ex. traduction / OCR).

6. Cookies et requêtes réseau

Certaines opérations (par exemple récupération d’images pour affichage ou traitement côté client) peuvent effectuer des requêtes réseau vers les URL que vous consultez, avec des options de type « credentials » selon le cas, afin de respecter les contraintes des sites visités. L’extension ne lit pas vos mots de passe stockés dans le navigateur et ne les transmet pas.

7. Pas de vente de données

Nous ne vendons pas vos données personnelles. Nous n’exploitons pas de profilage publicitaire au sein de l’extension et ne revendons pas l’historique de lecture à des tiers.

8. Vos choix

  • Vous pouvez supprimer les données de l’extension via les paramètres du navigateur (données de l’extension) ou les options prévues dans l’extension.
  • Vous pouvez retirer votre clé API Gemini à tout moment dans les paramètres.
  • Vous pouvez désinstaller l’extension : le stockage associé peut alors être effacé selon les réglages du navigateur.

9. Enfants

L’extension n’est pas destinée aux enfants de moins de 13 ans de manière spécifique ; le contenu des sites que vous visitez relève de votre responsabilité.

10. Modifications

Nous pouvons mettre à jour cette politique pour refléter l’évolution de l’extension ou des obligations légales. La date en tête de page sera ajustée en conséquence.

Rappel pour le Chrome Web Store : indiquez l’URL publique de cette page dans le champ « Politique de confidentialité » du tableau de bord développeur après l’avoir hébergée (par ex. GitHub Pages).
