Guide d'Utilisation MangaCentral

Découvrez toutes les fonctionnalités de votre assistant de lecture. MangaCentral a été conçu pour automatiser votre suivi de lecture sans jamais vous obliger à quitter vos sites préférés.

Une approche 100% Locale

MangaCentral fonctionne directement dans votre navigateur. L'extension ne contient aucun scan et ne nécessite aucune création de compte sur nos serveurs. Vos données vous appartiennent.

Installation & Épinglage

Pour une utilisation optimale, MangaCentral doit toujours être à portée de clic.

  1. Une fois l'extension installée depuis le Chrome Web Store, cliquez sur l'icône de pièce de puzzle (Extensions) en haut à droite.
  2. Trouvez "MangaCentral" dans la liste.
  3. Cliquez sur l'icône d'épingle à côté.

Ajouter un manga à sa bibliothèque

La bibliothèque vous permet de suivre vos œuvres (En cours, En pause, Terminés). Il existe deux méthodes pour y ajouter une série :

1. L'ajout Magique (Recommandé)

C'est la méthode la plus simple. MangaCentral analyse la page web sur laquelle vous vous trouvez pour extraire le titre et l'image de couverture.

  • Rendez-vous sur la page d'un chapitre du manga que vous souhaitez ajouter (sur votre site de scantrad préféré).
  • Ouvrez l'extension MangaCentral.
  • Dans l'onglet Librairie, cliquez sur le bouton 📄 Page actuelle.
  • L'extension remplira automatiquement le formulaire avec le titre et l'URL. Vous n'avez plus qu'à sauvegarder !

2. L'ajout Manuel

Si l'extension n'arrive pas à lire les informations du site, vous pouvez le faire manuellement.

  • Cliquez sur ✏️ Manuellement.
  • Saisissez le titre et collez l'URL de la page.
  • Vous pouvez optionnellement fournir l'URL d'une image de couverture, ou importer une image depuis votre ordinateur.

Le Panneau Flottant (Overlay)

Lorsque vous visitez un site de manga compatible, MangaCentral injecte un panneau flottant discret. Il s'agit de votre "télécommande" de lecture.

  • Détection Auto : En ouvrant un chapitre, l'overlay détectera le numéro du chapitre et l'enregistrera dans votre historique et votre bibliothèque.
  • Auto-Scroll : Démarrez un défilement automatique paramétrable (vitesse ajustable dans les options) pour lire sans toucher à la souris.
  • Navigation : Remontez tout en haut de la page ou passez au chapitre suivant d'un simple clic.

Synchronisation Cloud (AniList)

Bien que MangaCentral soit local, vous pouvez lier votre compte AniList pour sauvegarder votre progression dans le cloud.

  1. Allez dans les Paramètres (icône ⚙).
  2. Cliquez sur Se connecter avec AniList et autorisez l'application.
  3. Importer mes œuvres : Ce bouton rapatriera toute votre liste AniList directement dans MangaCentral.
  4. Synchronisation automatique : Une fois connecté, chaque fois que MangaCentral détecte un nouveau chapitre lu, il l'enverra discrètement à AniList.

Forcer la synchronisation

Si vous avez lu des chapitres hors ligne ou si un envoi a échoué, le bouton "Forcer la synchro AniList" mettra à jour votre profil en ligne avec vos données locales actuelles.

Traduction Assistée par IA (Google Gemini)

L'outil le plus puissant de MangaCentral vous permet de traduire des bulles de texte directement sur l'image.

  1. Dans les Paramètres, collez une clé d'API Google Gemini gratuite (obtenable via Google AI Studio).
  2. Sur un manga, ouvrez l'Overlay et cliquez sur l'icône de Traduction.
  3. Dessinez un rectangle englobant parfaitement le texte à traduire.
  4. L'IA analysera l'image, extraira le texte (OCR) et vous l'affichera traduit dans la langue choisie.

Attention à la sélection

Plus votre rectangle est précis (sans inclure de dessins autour de la bulle), plus l'IA sera performante pour extraire les caractères (surtout pour le Coréen ou le Japonais).

Sauvegardes (Export / Import)

Vous changez de PC ou vous réinstallez Chrome ? Ne perdez pas votre bibliothèque !

  • Dans les paramètres, utilisez Exporter mes données pour télécharger un fichier .json contenant toute votre bibliothèque, votre historique et vos sites personnalisés.
  • Utilisez Importer une sauvegarde sur votre nouveau navigateur pour tout restaurer instantanément.

Sites Personnalisés (Mode Avancé)

MangaCentral reconnaît nativement des dizaines de lecteurs (MangaTheme, Madara, etc.). Si vous tombez sur un site inconnu, l'Overlay n'apparaîtra pas et le suivi ne se fera pas. Vous pouvez régler cela vous-même !

Ajouter un site personnalisé

Dans les paramètres, sous "Sites personnalisés", vous pouvez définir comment l'extension doit "lire" le site :

  • Domaine : L'URL du site (ex: monsite.com).
  • Sélecteur titre (CSS) : L'élément HTML contenant le titre du manga (ex: h1.entry-title).
  • Pattern chapitre (Regex) : L'expression régulière pour extraire le numéro du chapitre de l'URL (ex: chapter-(\d+)).
  • Conteneur d'images : L'élément contenant les pages du manga, nécessaire pour l'auto-scroll.

Le bouton "Tester la page actuelle" vous permet de vérifier en direct si vos sélecteurs fonctionnent !

Désactivation Ciblée

MangaCentral s'injecte sur de nombreuses pages. Si l'extension crée un bug d'affichage sur un site précis (qui n'est pas un site de manga), vous pouvez la désactiver spécifiquement pour ce site.

  • Ouvrez l'extension sur le site posant problème.
  • Allez dans Paramètres > Activation.
  • À côté de "Onglet actuel", cliquez sur Désactiver ce site. L'extension ne s'exécutera plus jamais sur ce domaine.

Dépannage & Exceptions

Le web est chaotique. Voici pourquoi MangaCentral peut parfois sembler ne pas fonctionner.

1. L'historique / Le chapitre ne se met pas à jour

Les Menus Déroulants vs Changement d'URL

MangaCentral détecte que vous avez lu un chapitre lorsque l'URL de votre navigateur change (ex: de /chapter-1 à /chapter-2).

Certains sites utilisent un menu déroulant pour changer les images sans recharger la page ni changer l'URL. L'extension est "aveugle" à ce changement.

Solution : Si le site le permet, utilisez le bouton "Chapitre Suivant" de l'Overlay MangaCentral pour forcer le chargement, ou rechargez la page manuellement (F5).

2. Images cassées dans la bibliothèque (Erreurs 403)

Protections Anti-Hotlinking

Les serveurs des sites de scantrad coûtent cher. Pour éviter que d'autres applications n'utilisent leur bande passante, ils bloquent l'affichage des images (Erreur 403) si l'image est chargée hors de leur propre site.

Ce que fait l'extension : MangaCentral utilise des autorisations avancées (declarativeNetRequest) pour intercepter le chargement de l'image de couverture et dire au serveur "Hé, je suis bien sur ton site !".

Pourquoi ça échoue quand même : Certains sites utilisent des protections dynamiques impossibles à simuler. Dans ce cas, la couverture apparaîtra cassée.
Solution : Modifiez le manga manuellement et importez une image depuis votre ordinateur (bouton "Ou importer une image").
