Documentation MangaCentral

Bienvenue dans le guide complet d'utilisation de MangaCentral. Découvrez comment tirer le meilleur parti de votre assistant de lecture local.

Qu'est-ce que MangaCentral ?

MangaCentral est une extension de navigateur qui agit comme un "marque-page intelligent". Elle ne fournit aucun contenu piraté ni hébergement de scans. Son unique but est d'améliorer votre expérience sur les sites que vous visitez déjà.

Installation & Épinglage

Après avoir installé l'extension depuis le Chrome Web Store, il est fortement recommandé de l'épingler à votre barre d'outils pour y accéder rapidement.

  1. Cliquez sur l'icône de pièce de puzzle (Extensions) en haut à droite de Chrome.
  2. Trouvez "MangaCentral" dans la liste.
  3. Cliquez sur l'icône d'épingle à côté.
  4. L'icône de MangaCentral sera désormais toujours visible en haut à droite.

Le Tableau de Bord

Le tableau de bord est le cœur de MangaCentral. C'est ici que vous retrouvez toutes vos séries sauvegardées.

Ajouter un Manga

Il existe deux façons d'ajouter un manga à votre bibliothèque :

  • Automatique (Recommandé) : Cliquez sur "Ajouter", choisissez "Recherche API", tapez le nom du manga et sélectionnez une source (MangaDex, Jikan, AniList). Les informations (titre, couverture) seront remplies automatiquement. Collez simplement l'URL du site où vous le lisez.
  • Manuel : Saisissez vous-même toutes les informations et l'URL de l'image de couverture.

Statuts de Lecture

Vous pouvez organiser vos mangas avec différents statuts : En cours, En pause, ou Terminé. L'extension filtrera l'affichage en fonction de ces catégories.

Panneau Flottant (Overlay)

Lorsque vous visitez un site de manga, MangaCentral injecte un petit panneau flottant discret. Ce panneau offre plusieurs outils de confort.

  • Navigation Rapide : Boutons pour remonter tout en haut ou descendre tout en bas de la page.
  • Auto-Scroll : Défilement automatique de la page pour une lecture sans les mains.
  • Suivi de Chapitre : L'extension tentera de détecter automatiquement le numéro du chapitre actuel et de mettre à jour votre tableau de bord.

Traduction Assistée (IA)

MangaCentral intègre un outil expérimental de traduction propulsé par l'IA (Google Gemini). Cet outil extrait le texte d'une image (OCR) et le traduit en français.

Configuration

  1. Ouvrez le tableau de bord de MangaCentral.
  2. Allez dans les Paramètres (icône d'engrenage).
  3. Obtenez une clé API Gemini gratuite sur le Google AI Studio.
  4. Collez la clé dans le champ prévu et enregistrez.

Comment l'utiliser

Sur une page de lecture, cliquez sur l'icône de traduction dans le panneau flottant, puis dessinez un rectangle autour de la bulle de texte que vous souhaitez traduire.

Conseil de précision

Pour que l'IA fonctionne correctement, votre sélection doit englober uniquement le texte. Évitez de sélectionner des éléments de dessin ou de décor autour de la bulle, ce qui pourrait perturber l'extraction (OCR).

Dépannage & Exceptions (Pourquoi ça ne marche pas ?)

Étant donné la nature chaotique des sites de mangas, l'extension ne peut pas fonctionner parfaitement à 100% sur tous les sites du web. Voici les problèmes courants et leurs explications.

1. Le chapitre ne se met pas à jour dans l'historique

Normalement, MangaCentral détecte que vous changez de chapitre quand l'URL de la page web change. Cependant :

Le problème des Menus Déroulants (Dropdowns)

Certains sites (notamment ceux utilisant des lecteurs spécifiques) ne changent pas l'URL lorsque vous passez au chapitre suivant via un menu déroulant ou un bouton "Suivant" dynamique. L'extension ne "voit" donc pas le changement de page.

Solution : Rechargez la page (F5) ou utilisez les boutons du panneau flottant MangaCentral si disponibles.

2. Les couvertures ne s'affichent pas dans le Tableau de Bord

Vous avez ajouté un manga, mais au lieu de la couverture, vous avez une image cassée ou rien du tout.

Protections Anti-Hotlink (Erreurs 403 / CORS)

Les sites de mangas paient pour la bande passante. Pour éviter que d'autres sites n'utilisent leurs images, ils bloquent le chargement si l'image n'est pas demandée depuis leur propre site. C'est ce qu'on appelle la protection anti-hotlink.

Ce que fait MangaCentral : L'extension utilise des règles réseau (declarativeNetRequest) pour tenter de contourner ces protections localement dans votre navigateur en modifiant les en-têtes (Headers). Mais certains sites ont des protections très avancées que l'extension ne peut pas contourner.
Solution : Utilisez des images provenant d'API publiques (MangaDex, Jikan) via l'ajout automatique. Ces images ne sont pas bloquées.

3. L'Overlay n'apparaît pas du tout

Certains sites bloquent l'injection de scripts tiers ou utilisent des cadres (Iframes) complexes pour afficher leurs lecteurs. Sur ces sites, MangaCentral ne pourra pas injecter son panneau flottant ni suivre votre lecture automatiquement.

Données & Confidentialité

L'une des priorités de MangaCentral est le respect de votre vie privée.

  • Tout est local : Vos données de lecture, vos mangas sauvegardés et votre historique sont stockés exclusivement dans la mémoire locale de votre navigateur (chrome.storage.local).
  • Aucun compte requis : Nous n'avons pas de serveurs, pas de base de données en ligne. Vous ne pouvez donc pas "synchroniser" entre deux ordinateurs, mais vos données vous appartiennent à 100%.
  • Clé API Gemini : Si vous renseignez une clé, elle ne quitte jamais votre ordinateur (sauf pour interroger les serveurs de Google lors d'une demande de traduction explicite).

Lire la politique de confidentialité complète.
