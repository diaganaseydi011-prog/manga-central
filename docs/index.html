MangaCentral
Assistant de lecture pour manga / manhwa
Extension Chrome — suivi local & outils de confort

Un tableau de bord pour vos lectures

MangaCentral ne fournit pas de scans : il organise ce que vous lisez déjà, directement dans le navigateur.

  • •Bibliothèque : ajoutez vos titres avec URL, couverture, statut et dernier chapitre lu.
  • •Historique : les chapitres visités sur des sites compatibles sont mémorisés localement.
  • •Recherche : trouvez des titres via MangaDex, Jikan ou AniList, sans compte obligatoire.
  • •Panneau flottant sur la page : navigation haut/bas, auto-scroll et outil de sélection de zone.

Traduction assistée (optionnelle)

Si vous le souhaitez, vous pouvez configurer une clé API Google Gemini pour activer la traduction assistée d’une zone sélectionnée (OCR + traduction).

  • •Action explicite : rien n’est envoyé sans que vous cliquiez sur l’outil OCR / Traduire.
  • •Clé locale : la clé API saisie reste stockée dans chrome.storage.local sur votre appareil.
  • •Contrôle : vous pouvez supprimer la clé ou désinstaller l’extension à tout moment.

Conseil : pour de meilleures traductions, sélectionnez une zone qui contient uniquement la bulle ou le texte à traduire.
