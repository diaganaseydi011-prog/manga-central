3. DESCRIPTION DÉTAILLÉE (description complète du listing)
--------------------------------------------------------------------------------

(Copier le bloc ci-dessous tel quel, ou adapter légèrement.)

---

MangaCentral est un assistant de lecture pour bandes dessinées en ligne (manga, manhwa, webtoon). Il ne remplace pas les sites que vous utilisez : il ajoute des outils et un suivi local sur votre navigateur.

**Tableau de bord (icône de l’extension)**
• Vue d’ensemble de votre activité et accès rapide aux sections.
• **Bibliothèque** : liste des titres suivis, filtres, tri, ajout manuel (titre, URL, statut, couverture).
• **Découverte** : recherche de titres via des bases publiques (MangaDex, Jikan, AniList) ; si vous configurez une clé API Google Gemini, une suggestion de lien peut compléter les résultats.
• **Historique** : dernières lectures détectées sur les pages compatibles.
• **Paramètres** : vitesse de défilement par défaut, clé API Gemini (optionnelle), langue cible pour la traduction, sites personnalisés (sélecteurs CSS pour adapter l’extension à un lecteur non listé par défaut).

**Sur la page de lecture**
Lorsqu’un chapitre est reconnu, un panneau flottant apparaît : titre, numéro de chapitre, raccourcis haut/bas de page, défilement automatique réglable, et outil de **sélection de zone** pour envoyer une capture à l’API Gemini (traduction / transcription du texte visible dans la zone), uniquement si vous avez saisi une clé et que vous lancez l’action.

**Données et confidentialité**
Les données de bibliothèque et d’historique sont stockées **localement** sur votre appareil. La clé Gemini est stockée localement par vos soins. Les appels à Google ne sont effectués que lorsque vous utilisez la traduction. Détail complet : voir la politique de confidentialité liée ci-dessous.

**Mention importante**
Respectez les droits d’auteur et les conditions d’utilisation des sites que vous consultez. MangaCentral ne fournit pas de contenu protégé et ne constitue pas un service de téléchargement de chapitres.

6. DÉCLARATION DES DONNÉES (Data safety — lignes directrices)
--------------------------------------------------------------------------------

• Données collectées côté utilisateur : historique de lecture et bibliothèque (titres, chapitres, URL, couvertures éventuelles) — stockage **local** sur l’appareil.
• Données traitées par des tiers lorsque l’utilisateur agit : Google (API Gemini) pour traduction/capture si clé fournie ; MangaDex, Jikan, AniList pour métadonnées/recherche.
• Pas de vente de données personnelles (aligné sur la politique de confidentialité hébergée).
• Chiffrement : communications HTTPS vers les services ; stockage navigateur standard pour l’extension.
